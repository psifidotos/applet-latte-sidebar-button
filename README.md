# Requires

- Latte >= 0.10
- Plasma >= 5.12
- KDeclarative

# Install

This is a QML applet and as such it can be easily installed from Plasma 5 Widgets Explorer or alternative you can execute `plasmapkg2 -i .` in the root directory of the applet.
